# Clash Royale Chest Probability Tool

A web tool for checking Clash Royale chest opening probabilities and arena rewards.

## Features
- Multiple chest types (Normal, Magic, Season, Evolution, Hero)
- Opening probability analysis
- Reward amount query
- Reward probability tables
- Bilingual support (Chinese/English)

## Files
- `index.html` - Chinese version
- `en.html` - English version

## Deployment on GitHub Pages

1. Create a new GitHub repository
2. Upload both HTML files
3. Go to repository Settings > Pages
4. Select "Deploy from a branch" and choose main/master branch
5. Save - your site will be available at `https://[username].github.io/[repository-name]/`

## Usage
- Open `index.html` for Chinese version
- Open `en.html` for English version
- Click the language switch button to change between versions
